# Journal

Un bilan par jour (optionnel). 5 minutes le soir pour noter ce qui compte.

**Template** : Ctrl+T puis "Journal" dans Obsidian.

**Convention de nommage** : `YYYY-MM-DD.md`
Exemple : `2026-05-10.md`

**Conseil** : ne pas forcer la régularité. Mieux vaut 2 bilans par semaine bien écrits que 7 bilans vides.
