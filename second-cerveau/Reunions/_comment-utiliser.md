# Réunions

Une note par réunion. Créée après chaque rendez-vous, appel ou meeting.

**Template** : Ctrl+T puis "Réunion" dans Obsidian.

**Convention de nommage** : `YYYY-MM-DD - [Personnes] - [Sujet].md`
Exemple : `2026-05-10 - Carl Maxime - Point Versi Invest.md`

**Astuce** : tagger les participants avec `[[Contacts/Prénom Nom]]` pour créer les backlinks automatiquement dans le graphe.
