# Idées

Business, réflexions, opportunités. Tout ce qui mérite d'être capturé avant d'être oublié.

**Template** : Ctrl+T puis "Idée" dans Obsidian.

**Convention de nommage** : `YYYY-MM-DD - Titre.md`
Exemple : `2026-05-10 - Versi Capital structure possible.md`

**Conseil** : une idée par note. Courte ou longue, peu importe. Le but est de capturer, pas de rédiger.
