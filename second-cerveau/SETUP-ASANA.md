# Setup Asana dans Claude

Asana est un **connector natif Claude** depuis 2025-2026. Pas besoin de MCP technique, pas de Node.js, pas de token à générer.

## Setup (60 secondes)

1. Va sur **claude.ai/customize/connectors**
2. Cherche **Asana** dans le directory
3. Clique **"Connect"**
4. Autorise via le flow OAuth Asana (sélectionne ton workspace)
5. C'est connecté

## Test

Dans n'importe quelle conversation Claude :
- "Liste mes projets Asana"
- "Crée une tâche dans [projet] : [intitulé]"
- "Quelles sont mes tâches en retard ?"

## Plans requis

- **Directory connectors** (incluant Asana) : disponibles sur **tous les plans Claude**, y compris Free
- Sur Team/Enterprise : seul l'admin peut activer les connectors pour le workspace
