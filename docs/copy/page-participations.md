## Test persona — validation

**Test Leila** (apporteur d'affaires, 90 secondes) :
Leila scanne la page pour évaluer la solidité d'ISSA Capital avant de soumettre son dossier. Elle voit : Gradient One co-fondé en 2020 (pas une structure créée la semaine dernière), Versi Immobilier (marchand de biens — elle connaît le secteur), Immocrew (live, avec site), Versimo (live, avec site), immobilier résidentiel IDF en direct. Elle comprend que la holding investit vraiment dans l'immo. Son objection "capacité réelle d'investissement ?" est partiellement traitée par la structure (4 participations opérationnelles + immo direct). Elle est prête à soumettre. Verdict : la page remplit son rôle de validation de sérieux.

Verdict : page suffisamment détaillée pour Marc.
