## PARTIE 3 — Arbitrage 3 OCC F1 ambiguës

### OCC-6 — `mission/page.tsx` — Décision : REFORMULER

**Formulation actuelle** :
```
Ne pas confier le patrimoine familial à des logiques qui lui sont étrangères. Pas à un fonds à calendrier contraint, pas à des structures conçues pour être revendues.
```

**Verdict : REFORMULER**

**Justification éditoriale** : La première phrase est déjà affirmative et forte — "Ne pas confier le patrimoine familial à des logiques qui lui sont étrangères" dit la décision avec clarté. Les deux propositions suivantes ("Pas à un fonds... pas à des structures...") n'ajoutent pas de substance : elles précisent les adversaires au lieu d'affirmer l'identité. Dans le contexte "Décision fondatrice" de la page `/mission`, le ton peut être tranchant — mais le tranchant passe mieux par l'affirmative que par la liste des exclusions. La première phrase reformulée en affirmative suffit.

**old_string exact** (pour @fullstack) :
```
Ne pas confier le patrimoine familial à des logiques qui lui sont étrangères. Pas à un fonds à calendrier contraint, pas à des structures conçues pour être revendues.
```

**new_string exact** :
```
La famille Issa a choisi de garder le contrôle de son patrimoine — indépendant, privé, organisé selon ses propres convictions. L'horizon est celui des générations à venir.
```

### OCC-10 — `opportunites/page.tsx` — Décision : REFORMULER

**Formulation actuelle** :
```
Cette page est faite pour les apporteurs d'affaires et les fondateurs qui cherchent un partenaire capitalistique sérieux. Pas un fonds. Pas un comité d'investissement qui se réunit tous les 6 mois. Une holding familiale qui décide vite sur les dossiers qualifiés.
```

**Verdict : REFORMULER**

**Justification éditoriale** : La tension entre "qualification rapide pour ce persona" et règle P0 se résout ici par la reformulation. Les "Pas un fonds. Pas un comité." sont fonctionnellement utiles pour disqualifier la confusion — mais ils peuvent être remplacés par une formulation affirmative qui produit le même effet de qualification sans les négations. Le persona apporteur d'affaires lit entre les lignes : "une holding familiale sans comité trimestriel qui décide vite" contient déjà l'exclusion des fonds et des comités, de façon implicite et plus professionnelle.

**old_string exact** (pour @fullstack) :
```
Cette page est faite pour les apporteurs d&apos;affaires et les fondateurs qui cherchent un partenaire capitalistique sérieux. Pas un fonds. Pas un comité d&apos;investissement qui se réunit tous les 6 mois. Une holding familiale qui décide vite sur les dossiers qualifiés.
```

**new_string exact** :
```
Cette page s&apos;adresse aux apporteurs d&apos;affaires et aux fondateurs qui cherchent un actionnaire de long terme : une holding familiale, sans comité trimestriel, qui décide sur les dossiers qualifiés.
```
