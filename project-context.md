## Principe directeur #0 — VITRINE, PAS CONVERSION (verrouillé par Thomas 2026-04-07)

> **Citation exacte Thomas** : *"On est pas la pour plaire au prospect. On est la pour avoir une belle vitrine."*

### Ce que le site DOIT être
- Une **vitrine** premium qui raconte qui est la famille Issa, son héritage, son écosystème
- Une **présence institutionnelle** qui inspire le respect sans chercher à convaincre
- Un site qui **se suffit à lui-même** : si personne ne remplit jamais le formulaire, le site a quand même réussi

### Ce que le site N'EST PAS
- Une landing page optimisée conversion / un funnel / une machine à leads / un deck de pitch

### Calibration agents (1 règle par agent)
1. **@copywriter** : ton éditorial pas persuasif. Zéro tactique de copy de conversion (PAS/AIDA/BAB en sourdine uniquement).
2. **@design** : esthétique muséale/éditoriale. Pas de CTAs agressifs, pas de trust badges.
3. **@ux** : parcours contemplatifs. Friction OK si elle sert la posture premium.
4. **@growth** : pas d'AARRR/funnel/boucles virales. La conversion n'est pas le KPI.
5. **@seo** : branding et autorité, pas trafic transactionnel.
6. **@qa / testeurs** : mesurer perception de prestige / crédibilité / respect, pas conviction à soumettre.
7. **@fullstack** : analytics Umami uniquement (cookieless, open source). Pas de heatmaps ni session recording.

### Gates GP recalibrées pour ce projet
- **GP7** → "Respect inspiré" (pas conviction à s'inscrire)
- **GP9** → "Identité lisible" (pas outputs utiles)
- **GP10** → "Mémorabilité" (pas fidélisation)
